# think-inside-the-box

A capture the flag with the Vulnix VM from VulnHub

https://www.vulnhub.com/entry/hacklab-vulnix,48/


The project is divided into two parts

## Part A - Attack:
The goal is to capture the flag at `/root`.

1. **Reconnaissance**
2. **Gaining a Foothold**
3. **Privilege Escalation**
4. **Getting the flag!**


## Part B - Protect:
Harden the machine with a bash script and try to attack again to proof if the hardening works.


My solution is available in walkthrough.md / walkthrough.pdf and vulnix_fix.sh
